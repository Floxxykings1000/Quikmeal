# MealFlow - Smart Meal Planning App

![MealFlow Logo](assets/icons/icon-192x192.png)

MealFlow is a Progressive Web App (PWA) that helps you plan meals, manage your pantry, and reduce food waste through AI-powered suggestions.

## 🌟 Features

- **AI-Powered Meal Planning**: Generate weekly meal plans based on your preferences
- **Pantry Management**: Track ingredients and get expiration alerts
- **Smart Grocery Lists**: Auto-generated lists based on meal plans
- **Recipe Suggestions**: Get recipes based on ingredients you already have
- **Offline Support**: Full functionality without internet connection
- **PWA**: Install on any device (iOS, Android, Desktop)
- **Data Persistence**: All data saved locally with IndexedDB
