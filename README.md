## ✨ Features

- 🤖 **AI Meal Planning** - Generate personalized weekly meal plans
- 📦 **Pantry Management** - Track ingredients with expiration alerts
- 🛒 **Smart Grocery Lists** - Auto-generated from meal plans
- 🍳 **Recipe Suggestions** - Use ingredients before they expire
- 📱 **PWA Support** - Install on iOS, Android, and Desktop
- 🔄 **Offline First** - Full functionality without internet
- 💾 **Local Storage** - Your data stays on your device
- 🎨 **Beautiful UI** - Clean, modern interface with dark mode ready

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Floxxykings1000/Quikmeal.git

# Navigate to project
cd Quikmeal

# Serve locally
npx serve .
# or
python3 -m http.server 8080
```
